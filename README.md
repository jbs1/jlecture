jlecture
========

Lecture Database for Jacobs  

url:  
http://jlecture.user.jacobs-university.de/  


to push changes to the webserver,  
you need to simply execute
```
./sync
```


Sorted by Courses/ID:  
Courses can have different versions of different years  


Features:
*Video Lecture
*Slides
*Lecture Notes
*Homework/Assigments
*Exams
*Solutions
